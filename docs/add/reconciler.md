# 对账员

## 概述

核对银行流水与 Beancount 账本的差异，生成调账建议。

## 输入输出

| 输入 | 输出 |
|------|------|
| 银行流水 (CSV) | 差异列表 |
| Beancount 账本 | 调账建议 |

## 流程

1. 加载银行流水（CSV）
2. 加载账本交易（Beancount loader）
3. 自动匹配（日期窗口 + 金额容差 + 打分）
4. LLM 分析未匹配项
5. 生成调账建议

## 匹配策略

- 日期窗口：±2 天
- 金额阈值：±1 元
- 打分排序：综合日期差、金额差

## 验证

- 自动匹配：规则驱动
- LLM 匹配：语义分析

## 演进

v1 → v2：
- Beancount 解析：正则 → loader
- 匹配算法：精确 → 窗口+容差+打分
- LLM：普通 → JSON 模式
- 结果：ReconciliationReport
