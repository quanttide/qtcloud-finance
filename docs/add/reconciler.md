# 对账员

## 做什么

核对银行流水和账本的差异，找出对不上的交易。

## 流程

1. **加载**：读取银行流水 CSV 和账本交易
2. **匹配**：按日期（±2天）和金额（±1块）自动配对
3. **分析**：配不上的交给 LLM 语义分析
4. **输出**：生成调账建议

## 例子

银行有一条 3月28日 工资 15000，账本里没有。

```
1. 加载：银行1条，账本5条
2. 匹配：4条自动配上，1条银行没配上
3. 分析：LLM 识别这条是工资，建议补记
4. 输出：
   建议补记：
   2026-03-28 * "3月工资"
     Expenses:Salary:Wages   15000.00 CNY
     Assets:Bank:Company  -15000.00 CNY
```
