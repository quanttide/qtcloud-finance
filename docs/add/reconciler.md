# 对账员

## 概述

核对银行流水和账本的差异，找出对不上的交易。

## 场景

**场景：月底对账**

银行流水有一条：
```
2026-03-28  工资  15000  公司账户
```

账本里没有这条记录。

对账员找出差异，生成建议：
```
建议补记：
2026-03-28 * "3月工资"
  Expenses:Salary:Wages   15000.00 CNY
  Assets:Bank:CompanyAccount  -15000.00 CNY
```

**场景：金额对不上**

银行流水：午餐 15.00
账本记录：午餐 12.00

差 3 块钱。可能是手续费，对账员会标出来让人确认。

## 匹配

自动匹配规则：
- 日期：前后 2 天内
- 金额：相差不超过 1 块

匹配不上的，交给 LLM 分析语义相似度。

## 验证

- 自动匹配：规则驱动，准确但死板
- LLM 匹配：语义分析，灵活但可能错
