# 记账员

## 概述

将自然语言描述的交易记录为 Beancount 格式。

## 输入输出

| 输入 | 输出 |
|------|------|
| 自然语言描述 | Beancount 记账 |

## 流程

1. 获取账本中的账户列表
2. 构建提示词（系统 + 用户）
3. 调用 LLM 生成 Beancount
4. 提取 Beancount 代码
5. 验证语法（Beancount 解析器）
6. 写入账本

## 验证

- 语法验证：使用 beancount.parser
- 借贷平衡
- 账户存在

## 设计原则

> 让大模型做打字员，让解析器做审核员，让代码保持简单
