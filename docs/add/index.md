# 架构设计文档

## 领域建模

### 领域实体

| 对象 | 说明 |
|------|------|
| 银行流水 | 银行导出的交易记录 (CSV) |
| Beancount 账本 | 复式记账格式文件 |
| 调账记录 | 确认后的调账分录 |

### 领域服务

| 智能体 | 职责 | 输入 | 输出 |
|--------|------|------|------|
| **记账员** | 记录交易 | 自然语言 | Beancount 记账 |
| **对账员** | 核对差异 | 银行流水 + 账本 | 差异 / 调账建议 |

### 上下文

```
┌─────────────┐    记账    ┌─────────────┐
│  自然语言   │ ─────────► │   记账员    │
└─────────────┘            └──────┬──────┘
                                 │ Beancount
                                 ▼
┌─────────────┐    对账    ┌─────────────┐    差异/建议    ┌─────────────┐
│  银行流水   │ ─────────► │   对账员    │ ────────────► │  人工确认   │
└─────────────┘            └──────┬──────┘                └──────┬──────┘
                                 │ Beancount                      │ 调账记录
                                 ▼                               ▼
                          ┌─────────────┐              ┌─────────────┐
                          │  Beancount  │ ◄─────────── │   账本      │
                          │    账本    │              └─────────────┘
                          └─────────────┘
```

## 技术实现

### 技术栈

- **Python** - 开发语言
- **Textual** - TUI 框架
- **Pydantic Settings** - 配置管理
- **Beancount** - 账本解析与验证
- **Ollama** - 本地 LLM

### 存储

- 本地文件
- 阿里云 OSS
- AWS S3

### 配置

环境变量：`ALIBABA_CLOUD_ACCESS_KEY_ID`、`OSS_ENDPOINT`、`OSS_BUCKET_NAME` 等

详见 [.env.example](../../.env.example)
