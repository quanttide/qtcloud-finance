# qtcloud-finance 开发文档

## 概述

qtcloud-finance 是一套财务 AI Agent 系统。

### 参与对象

| 对象 | 说明 |
|------|------|
| 银行流水 | 银行导出的交易记录 (CSV) |
| Beancount 账本 | 复式记账格式文件 |
| 调账记录 | 确认后的调账分录 |

### 智能体

| 智能体 | 职责 | 输入 | 输出 |
|--------|------|------|------|
| **记账员** | 记录交易 | 自然语言 | Beancount 记账 |
| **对账员** | 核对差异 | 银行流水 + 账本 | 差异 / 调账建议 |

### 流程

```
┌─────────────┐    记账    ┌─────────────┐
│  自然语言   │ ─────────► │   记账员    │
└─────────────┘            └──────┬──────┘
                                 │ Beancount
                                 ▼
┌─────────────┐    对账    ┌─────────────┐    差异/建议    ┌─────────────┐
│  银行流水   │ ─────────► │   对账员    │ ────────────► │  人工确认   │
└─────────────┘            └──────┬──────┘                └──────┬──────┘
                                 │ Beancount                      │ 调账记录
                                 ▼                               ▼
                          ┌─────────────┐              ┌─────────────┐
                          │  Beancount  │ ◄─────────── │   账本      │
                          │    账本    │              └─────────────┘
                          └─────────────┘
```
